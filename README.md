# Where is my room?

방을 보러 다닐 때 체크해야 할 사항들을 기록하는 애플리케이션입니다.

Where is my room?은 다음과 같은 항목을 제공합니다.

###### (*표시된 항목은 필수 입력 사항입니다.)

1. 보증금*
2. 월세*
3. 관리비*
4. 관리비 항목
5. 사진
6. 인근 지하철역
7. 건물 층 수, 해당 층 수
8. 방향
9. 방 구조
10. 크기
11. 옵션
12. 반려동물
13. 엘리베이터
14. 주차
15. 상세정보
